# Course-management
IDP Training Management System

> **Instituto de Desarrollo Profesional** · Ministerio de Educación Pública de Costa Rica

Sistema integral para la gestión de cursos, matrículas, mediadores y reportes académicos del IDP.  
Desarrollado con **FastAPI (Python)** en el backend y **React (Vite)** en el frontend, con autenticación JWT y base de datos en **Supabase (PostgreSQL)**.

---

## 🚀 Tecnologías

| Capa       | Tecnologías |
|------------|-------------|
| Backend    | FastAPI, Python 3.11+, Supabase (PostgreSQL), JWT, bcrypt, SlowAPI |
| Frontend   | React 18, Vite, React Router, CSS Modules / vanilla |
| Infra      | Passenger WSGI (producción), Uvicorn (desarrollo) |
